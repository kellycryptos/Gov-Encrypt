# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**DO NOT** open public issues for security vulnerabilities.

If you discover a vulnerability regarding:
- Private vote leakage
- Delegation graph deanonymization
- MPC circuit correctness
- Smart contract access control

Please email **security@kellycryptos.com** (or contact @kellycryptos via DM).

We appreciate your responsible disclosure and will acknowledge receipt within 24 hours.
